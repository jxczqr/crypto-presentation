Альткоины

Это альтернативные коины – любые криптовалюты со своим блокчейном (кроме биткоина). Некоторые из них похожи на биткоин. Другие ориентированы на внедрение и использование новых инструментов, а также расширение возможностей.

При помощи изменения открытого исходного кода ВТС, разработчики альткоинов могут ускорять транзакции, оптимизировать процесс майнинга, создавать различные автоматизированные контракты, формировать базу для работы с криптоприложениями и т.д.
